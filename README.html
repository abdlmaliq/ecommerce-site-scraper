What It Does

🔍

Form Search

Types your query into Jumia's search bar and hits Enter — exactly like a real user browsing the site.

🍪

Popup Handler

Auto-dismisses cookie banners, newsletter modals, and app-install prompts that would block scraping.

📜

Lazy Scroll

Scrolls incrementally down each page to trigger lazy-loaded product images and dynamically injected cards.

📄

Pagination

Automatically clicks "Next Page" and scrapes every result page up to your configured limit.

💰

Price & Discount

Captures current price, original (crossed-out) price, and the discount percentage label per product.

⭐

Ratings & Reviews

Extracts star rating and total number of customer reviews for each product listing.

🏷️

Trust Badges

Detects Jumia Express, Official Store, and other seller trust badges displayed on listings.

📦

JSON Output

Saves results with a full metadata block: query, timestamp, pages scraped, and total product count.

# For each page: scroll → extract → paginate
while page <= max_pages:
    scroll_page(driver)          # trigger lazy-loaded cards
    dismiss_popups(driver)       # in case one appeared mid-scroll

    products = parse_products(driver)
    all_products.extend(products)
    print(f"   Page {page}: {len(products)} products")

    if not go_to_next_page(driver):
        break              # no more pages
    page += 1

# Popup dismissal tries multiple selectors
def dismiss_popups(driver):
    selectors = ["button[data-testid='cookies-accept']",
                 ".-close", ".-dismiss", ".modal-close"]
    for sel in selectors:
        try:
            WebDriverWait(driver,2).until(
                EC.element_to_be_clickable((By.CSS_SELECTOR, sel))
            ).click()
        except: pass   # not present — move on

# Each product card parsed safely (no crash on missing fields)
def safe_text(card, selector, attr=None):
    try:
        el = card.find_element(By.CSS_SELECTOR, selector)
        return el.get_attribute(attr) if attr else el.text.strip()
    except NoSuchElementException:
        return None

Get Started

1

Install dependencies

webdriver-manager handles ChromeDriver automatically — no manual downloads needed.

pip install selenium webdriver-manager
2

Run with defaults

Searches "laptops" across up to 10 pages in silent headless mode.

python jumia_scraper.py
3

Custom query & page limit

Search any product. Use --visible to watch Chrome work in real time — great for debugging.

python jumia_scraper.py --query "gaming laptop" --pages 3
python jumia_scraper.py --query "dell laptop" --visible
4

Configure multi-query runs

Edit config.py to set multiple search terms, filters (min rating, max price), and output settings.

nano config.py
5

Read your results

Output lands in data/jumia_laptops.json — open in any JSON viewer or load in Python.

python -c "import json; d=json.load(open('data/jumia_laptops.json')); print(d['meta'])"
